BackgroundRandomizer 0.11
========================

Jquery plugin that randomize background color or background images of Divs.

Usage: 

	$("body").randomizza({
		color: ['color1', 'color2', 'color3'], 
		time: 400,
		images: ['url1', 'url2', 'url3']
	});

Plugin will work with every div with *random* class.
